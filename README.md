**# Spotify Clone**

A responsive Spotify Clone web player built using **HTML, CSS, and JavaScript**. This project mimics the core layout and behavior of Spotify with a stylish UI and smooth interactions.


**🚀 Features**

- Responsive Music Player
Plays music already saved in the library.

- Crazy UI
Stylish and interactive layout inspired by Spotify's dark theme.

- Fully Functional Controls
Includes play/pause, skip, volume control, progress bar, and other action buttons.

- Responsive Design
Works seamlessly across desktop, tablet, and mobile devices.
